# Doceus

**Modern, Open Source Medical Practice Platform**  
Zero-knowledge encryption. You own your data.

[AGPLv3 Licensed](LICENSE)

---

## Tech Stack

- **Laravel 12+**
- **PostgreSQL 17+**
- **Filament 4**

The `composer.json` file lists Laravel 12 and Filament v4 as core dependencies.
